Objetivo

Este guia visa estruturar o desenvolvimento do aplicativo SOS Unimed Vidas, organizando as etapas, ferramentas, e diretrizes técnicas.

Fases do Projeto

  • 🔹 Planejamento Estratégico
  • 🔹 Configuração de Ambiente e Repositórios
  • 🔹 Desenvolvimento da Interface (UI)
  • 🔹 Integração com Backend
  • 🔹 Testes e Ajustes Finais

Ferramentas Utilizadas

  • ✅ GitHub e Vercel para CI/CD
  • ✅ Figma (UI/UX)
  • ✅ Formspree (Formulário inicial)
  • ✅ OpenAI GPTs para geração de conteúdo e otimização

Submenus Atuais

  • 🚑 Emergência Médica
  • 📞 Contato com Plano
  • 📍 Localização de Unidades
  • 🧠 Suporte Emocional
  • 📄 Termos e Condições

Fluxo Padrão de Deploy

  1. Commit no GitHub
  2. Deploy automático via Vercel
  3. Verificação manual de UI

Observações Finais

O guia é vivo e pode ser atualizado conforme evolução técnica e decisões com stakeholders.
