🌐 Visão Geral do Projeto

O SOS Unimed Vidas é uma plataforma desenvolvida para auxiliar os beneficiários da Unimed Cachoeiro do Itapemirim no acesso rápido a serviços de saúde emergenciais.

Principais Características:

  • Acesso rápido a serviços médicos
  • Integração com prontuário eletrônico
  • Localização de unidades de saúde próximas
  • Chat de emergência com profissionais
  • Histórico de atendimentos

🏛️ Arquitetura do Sistema

Diagrama de Componentes ▼

Estrutura em Camadas

  1. Frontend: Aplicação React.js
  2. Backend: API Node.js
  3. Banco de Dados: Firebase Firestore
  4. Autenticação: Firebase Auth
  5. Hospedagem: Vercel
Fonte: Documentação Técnica do Projeto
Fluxo de Dados ▼

Sequência Típica

  1. Requisição do cliente (React)
  2. Processamento pelo backend (Node.js)
  3. Acesso ao banco de dados (Firebase)
  4. Resposta ao cliente

💻 Tecnologias Utilizadas

Frontend ▼
  • React.js v18

  • Material-UI v5
  • Axios para requisições HTTP
Backend ▼
  • Node.js v16
  • Express.js

🛠️ Configuração do Ambiente

1. Pré-requisitos ▼

Softwares Necessários

  1. Node.js v16+
  2. NPM v8+
  3. Git
  4. Editor de código (VS Code recomendado)
2. Clonar Repositório ▼

Passo a Passo

  1. Abrir terminal
  2. Executar: git clone https://github.com/Tomgit-cyber/guiaweb-sos
  3. Navegar até a pasta: cd guiaweb-sos
3. Instalar Dependências ▼

Comandos Necessários

  1. Frontend: cd frontend && npm install
  2. Backend: cd backend && npm install

🖥️ Desenvolvimento Frontend

Estrutura de Pastas ▼

Principais Diretórios

  • src/components/ - Componentes reutilizáveis
  • src/pages/ - Páginas da aplicação
  • src/services/ - Conexão com API
  • src/styles/ - Estilos globais
Executando o Frontend ▼

Passo a Passo

  1. Navegar até a pasta frontend: cd frontend
  2. Iniciar servidor de desenvolvimento: npm start
  3. Acessar no navegador: http://localhost:3000
