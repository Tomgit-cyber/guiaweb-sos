❓ Solução: Comando 'flutter' não reconhecido

Cenário do Erro

Você abre o terminal e recebe a mensagem:

'flutter' não é reconhecido como um comando interno
ou externo, um programa operável ou um arquivo em lotes.

Causa Principal

O sistema não sabe onde encontrar o Flutter. Isso ocorre porque o caminho para a pasta `bin` do SDK não foi adicionado corretamente às Variáveis de Ambiente do Windows.

✅ Como Resolver

  1. Confirme o Caminho: Verifique se a pasta `bin` está dentro do local onde você extraiu o Flutter (ex: C:\src\flutter\bin).
  2. Corrija as Variáveis de Ambiente: Use o painel sysdm.cpl para adicionar o caminho completo da pasta `bin` à variável `Path` do sistema.
  3. Reinicie o Terminal: Feche TODAS as janelas do terminal e abra uma nova para que a mudança tenha efeito.
